# Changelog

## 0.1.0

Initial open-source release.

- Config-driven agents defined in markdown and JSON
- Multi-provider LLM support (Anthropic, OpenAI, Google Gemini, AWS Bedrock, Azure OpenAI)
- Skills, knowledge base, and tool system
- Persistent stores (PGlite and PostgreSQL)
- HTTP runtime with SSE streaming and session management
- React components and embeddable chat widget
- CLI with 15+ commands (init, dev, chat, deploy, eval, connect, and more)
- Evaluation framework with LLM-judged assertions
- Security infrastructure (audit logging, field scrubbing, role-based access)
- MCP server support
